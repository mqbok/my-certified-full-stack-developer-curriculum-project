Introduction

Python is a beginner-friendly, high-level programming language known for its clean syntax and readability. It supports multiple programming paradigms, including object-oriented, procedural, and functional programming.

Python is widely used in fields like web development, data science, automation, and artificial intelligence.

Hello World

The simplest Python program is printing text to the screen. Here's how to do it :

print("Hello, World!")

print() is a built-in function that outputs text.

Text must be inside quotes ( " " or ' ' )

Input and Output

You can use print() to display output and input() to get input from the user :

# Output
print("Welcome!")

# Input
name = input("What is your name? ")
print("Hi, " + name + "!")

Note: input() always returns a string.

Variables

Variables are used to store data in Python. You create a variable by assigning a value using the = sign:

name = "Alice"
age = 25
is_active = True

Python automatically detects the data type.

Variable names should be lowercase with underscores (user_name).

Data Types

Python has several basic data types:

  • String (str) - Used to store text.
  • Integer (int) - Whole numbers, without decimals.
  • Float (float) - Numbers with decimal points.
  • Boolean (bool) - Stores either True or False.
  • List - An ordered collection of items (can be mixed types).
  • Tuple - Like a list, but unchangeable (immutable).
  • Dictionary (dict) - Stores data in key-value pairs.
Control Flow

Control the logic of your program with if, elif, and else :

age = 18

if age >= 18:
    print("You are an adult.")
elif age > 12:
    print("You are a teenager.")
else:
    print("You are a child.")

Python uses indentation to define code blocks (usually 4 spaces).

Functions

Functions allow you to group code into reusable blocks. They help make your code modular, readable, and easier to debug.

Defining a Function :

def greet(name):
print(f"Hello, {name}!")

Calling a Function:

greet("Alice")
# Output: Hello, Alice!

Use def to define a function

Use return to return values

Functions can have parameters and default values

Lists and Tuples

Lists are ordered, mutable collections.

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0])
# Output: apple

Tuples are ordered, immutable collections.

coordinates = (10, 20)
print(coordinates[1])
# Output: 20

Lists: mutable (append, remove, etc.)

Tuples: immutable and often used for fixed data

Loops

For Loop :

for i in range(5):
    print(i)

While Loop :

count = 0
while count < 5:
    print(count)
count += 1

Loop Control:

  • break: exits the loop
  • continue: skips to next iteration
  • else: optional block if the loop wasn't exited by break
Dictionaries

A dictionary is an unordered, mutable collection of key-value pairs.

person = {
    "name": "Alice",
    "age": 30,
}
print(person["name"])
# Output: Alice
